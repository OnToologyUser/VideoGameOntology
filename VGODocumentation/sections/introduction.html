1. Introduction back to ToC

This document presents the Video Game Ontology, an ontology created for describing knowledge of Video Games. The ontology focuses on modelling events happening inside video games as well as players and their playing behaviour.

The motivation for creating this ontology is to enable interoperability between different games, allowing them to exchange information between each other. For example, the Video Game Ontology could enable mapping characters, items or events between games. In addition, this ontology enables capturing deeper knowledge of player behaviour in video games, than what is currently being gathered with more traditional approaches.

This document specifies the classes, properties and their relations of the Video Game Ontology. The ontology is currently used in Gamecloud project implemented at Lappeenranta University of Technology, Finland. More information about the Gamecloud project can be found from the project official website
